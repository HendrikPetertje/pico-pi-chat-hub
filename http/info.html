What is this?

You are connected to PevaPub, a tiny wireless server running on a Raspberry Pi Pico W2, a microcontroller roughly the size of a stick of gum. There is no internet here. The Pico itself is the internet, for as long as you are connected to its WiFi network.

What can it do?

Right now it hosts a small chat room where everyone on the same WiFi can send short messages to each other. Over time it will also host simple games you can play together, all served from the same little chip.

Please be gentle

The hardware is about as minimal as it gets: a single-core processor running at 133 MHz with 264 KB of RAM. It is very easy to overwhelm. Too many requests at once, a large file, or a burst of messages can slow it to a crawl or crash it entirely.

On top of that, everything runs over open, unencrypted WiFi and plain HTTP. There are no passwords, no TLS, no privacy. Anyone nearby can see what you send.

The spirit of the thing

This is a small, fragile, handmade thing. Please be kind to each other and treat the server with care. Use it for what it is (a bit of fun) and resist any urge to push its limits or break things. The more gently everyone uses it, the better the experience for everyone.

Have fun!
